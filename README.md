## 🧠 Core Logic

The platform uses a **weighted scoring system** to evaluate tools:

- **Rating** → 50% weight  
- **Freshness Score** → 30% weight  
- **Description Quality** → 20% weight  

### 📊 Final Score Formula

Score = (Rating × 2 × 0.5) + (Freshness × 0.3) + (Description × 0.2)


This ensures a balanced and automated evaluation of all tools.

---

## 🏗️ System Architecture

The system follows a **three-layer architecture**:

### 1. Frontend Layer (Softr)
- Displays tools in structured format  
- Provides search and filtering  

### 2. Backend Layer (Airtable)
- Stores data of 250+ AI tools  
- Manages structured fields  

### 3. Logic Layer (Formula-based)
- Computes scores automatically  
- Eliminates manual work  

---

## 🔄 Data Flow

User → Frontend (Softr) → Backend (Airtable) → Logic (Formulas) → Display Results

---

## 🛠️ Tech Stack

- **Frontend:** Softr  
- **Backend:** Airtable  
- **Logic:** Formula-based computation
